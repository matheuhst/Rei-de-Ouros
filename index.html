Jogo de Ouros

Regras do jogo

Regra base

Cada jogador escolhe um número de 0 a 100. O sistema calcula a média de todos os números e multiplica por 0,8. Quem ficar mais perto do resultado vence a rodada.

Pontuação

Quem não vencer perde 1 ponto. Ao chegar em -10 pontos, o jogador é eliminado.

Primeira regra adicional

Os jogadores que escolherem números repetidos se tornarão inválidos, descontando 2 pontos de cada jogador.

Segunda regra adicional

Se algum jogador acertar exatamente o alvo, a penalidade dos outros jogadores é dobrada.

Duelo final

Se algum jogador escolher 0, o jogador que escolher o número 100 é o vencedor.
